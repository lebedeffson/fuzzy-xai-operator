# Формальные теоремы для глав 2 и 3

Цель файла - усилить математическую часть диссертации без разрыва с текущей реализацией. Теоремы ниже опираются на уже введённые объекты: `E_k`, `ExplainPlan`, `d_E`, `L(E)`, `I(E)`, `A_k^F`, `Delta`, `P_sit`, `F_ML`.

## Глава 2. Системный оператор

### Теорема 2.1. Существование и единственность композиционного объяснения

**Куда вставить:** после определения композиции, раздел 2.7 или 2.7.1.

Пусть `E_i` и `E_j` - объяснительные объекты соседних компонентов `C_i -> C_j`. Предположим:

1. задано допустимое согласование интерфейсов `T_ij`, переводящее словарь, правила, активации, неопределённость и след `E_i` в интерфейс `E_j`;
2. отображение `T_ij` детерминировано на своей области определения;
3. `d_E(T_ij(E_i), E_j) <= gamma_max`, где `gamma_max` задан в `ExplainPlan`;
4. функционал потери удовлетворяет оценке композиции

\[
L(E_j\odot T_{ij}(E_i))\le L(E_i)+L(E_j)+\kappa\gamma_{ij}.
\]

Тогда композиционное объяснение

\[
E_{ij}=E_j\odot T_{ij}(E_i)
\]

существует, единственно и является объяснительным объектом объединённого перехода `C_i -> C_j`.

**Доказательство.** По условию `T_ij(E_i)` определён и имеет интерфейс, совместимый с `E_j`. Неравенство `d_E <= gamma_max` означает, что переход лежит в допустимой области частичной операции `odot`; значит, композиция существует. Единственность следует из детерминированности `T_ij` и детерминированного правила объединения полей `L, R, alpha, u, tau`. Оценка для `L` является условием допустимой композиции и переносится на `E_ij`. Следовательно, `E_ij` корректен как объект объяснения объединённого компонента. ∎

### Теорема 2.2. `d_E` является псевдометрикой

**Куда вставить:** после введения меры рассогласования, раздел 2.8.

Пусть

\[
d_E(E_a,E_b)=\sum_s \beta_s d_s(E_a,E_b),\qquad \beta_s>0,\quad \sum_s\beta_s=1.
\]

Если все частные компоненты `d_s` являются псевдометриками на соответствующих полях объяснительного объекта, то `d_E` является псевдометрикой:

\[
d_E(E,E)=0,
\]

\[
d_E(E_a,E_b)=d_E(E_b,E_a),
\]

\[
d_E(E_a,E_c)\le d_E(E_a,E_b)+d_E(E_b,E_c).
\]

Если дополнительно нулевое значение компонентной меры следа означает совпадение существенных полей `tau`, то `d_E` становится метрикой на фактор-пространстве объяснений по несущественным метаданным.

**Доказательство.** Нулевая диагональ и симметрия следуют из тех же свойств всех `d_s` и положительности весов. Для треугольника:

\[
d_E(E_a,E_c)=\sum_s\beta_s d_s(E_a,E_c)
\le \sum_s\beta_s(d_s(E_a,E_b)+d_s(E_b,E_c))
=d_E(E_a,E_b)+d_E(E_b,E_c).
\]

Усиление до метрики получается факторизацией по отношению эквивалентности `E_a ~ E_b`, если `d_E(E_a,E_b)=0`. ∎

### Теорема 2.3. Оценка потери для последовательной композиции

**Куда вставить:** конец раздела 2.9, после оценки для композиционного графа.

Рассмотрим цепочку компонентов

\[
C_1\to C_2\to\dots\to C_n.
\]

Пусть для каждого перехода выполнены условия теоремы 2.1. Тогда для частичных композиций `E^(k)` справедлива оценка

\[
L(E^{(k+1)})\le L(E^{(k)})+L(E_{k+1})+\kappa\gamma_{k,k+1}+\Delta_{k,k+1}.
\]

Следовательно,

\[
L(E^{(n)})\le \sum_{k=1}^{n}L(E_k)+\kappa\sum_{k=1}^{n-1}\gamma_{k,k+1}+\sum_{k=1}^{n-1}\Delta_{k,k+1}.
\]

Для конечного конвейера потеря ограничена конечной верхней оценкой. Для бесконечной цепи предел конечен при условии сходимости правой части, то есть если ряды компонентных потерь, рассогласований и редукционных потерь суммируемы.

**Доказательство.** Индукция по `k`. База `k=1` очевидна. Переход `k -> k+1` следует из оценки теоремы 2.1 с добавлением редукционной потери `Delta`. Суммирование неравенств даёт верхнюю оценку. Условие для бесконечного случая является стандартным критерием ограниченности частичных сумм неотрицательного ряда. ∎

## Глава 3. Иерархия нечётких представлений

### Теорема 3.1. Полнота и минимальность иерархии представлений

**Куда вставить:** раздел 3.6 вместо слабого утверждения о покрытии.

Пусть `U_XAI` - конечное множество типов неопределённости, а `F_XAI` - конечная иерархия классов представлений с отношением покрытия `Cov(F)`. Предположим:

1. для каждого `u in U_XAI` существует хотя бы один класс `F`, такой что `u in Cov(F)`;
2. классы упорядочены по выразительности;
3. для каждого разделяющего типа существует класс, покрывающий его без избыточной структуры.

Тогда для каждого `u in U_XAI` существует минимальный класс

\[
F_u=\min\{F\in F_{XAI}:u\in Cov(F)\}.
\]

Если два минимальных класса покрывают один и тот же тип, они изоморфны относительно вложения представлений. Удаление любого базового класса, покрывающего хотя бы один разделяющий тип, нарушает полноту покрытия.

**Доказательство.** Множество покрывающих классов непусто и конечно, значит содержит минимальный элемент. Если существуют два различных минимальных элемента для одного типа, то по определению минимальности ни один не строго проще другого; следовательно, они эквивалентны по покрываемой структуре и изоморфны относительно вложения. Удаление класса, единственного минимального для некоторого разделяющего типа, оставляет этот тип либо непокрытым, либо покрытым только избыточным классом, что нарушает минимальность. ∎

### Теорема 3.2. Аппроксимационная оценка редукции

**Куда вставить:** раздел 3.7 после определения редукций и `Delta`.

Пусть `F_b` - более выразительный класс, `F_a` - более простой класс, заданы редукция

\[
\Pi_{b\to a}:F_b\to F_a
\]

и вложение

\[
\iota_{a\to b}:F_a\to F_b.
\]

Пусть мера различия `D_b` ограничена сверху единицей, а вложение не увеличивает расстояния. Тогда для любого `A in F_b`

\[
D_b(A,\iota_{a\to b}(\Pi_{b\to a}(A)))\le \varepsilon(A),
\]

где `epsilon(A)` конструктивно вычисляется из параметров объекта: полуширина интервала для `IntervalFS`, максимальное отклонение экспертных оценок от агрегата для `HesitantFS`, конфликтная компонента для `NeutrosophicFS`.

Если `epsilon(A) <= delta`, то редуцированный объект является `delta`-аппроксимацией исходного.

**Доказательство.** Для каждого базового класса редукция задаёт конкретный агрегат: середину/границу интервала, среднее/медиану экспертного множества, агрегат нейтрософских компонент. Подстановка этих выражений в `D_b` даёт соответствующую верхнюю оценку `epsilon(A)`. Ограниченность `D_b` и липшицевость вложения гарантируют, что обратное вложение редуцированного объекта не удаляется от исходного более чем на `epsilon(A)`. ∎

### Теорема 3.3. Завершимость итеративной редукции многоуровневого представления

**Куда вставить:** раздел 3.18 или новый раздел 3.19 после синтеза `F_ML`.

Пусть `A_ML` имеет конечное число уровней `m`, а каждый уровень принадлежит конечной иерархии глубины не более `N`. На каждом шаге алгоритм либо упрощает один уровень, не нарушая порог `Delta_max`, либо останавливается.

Тогда процесс завершается не более чем за `mN` шагов. Финальное представление нельзя дополнительно редуцировать без превышения порога потери.

**Доказательство.** Каждое допустимое упрощение строго уменьшает ранг хотя бы одного уровня в конечной иерархии. Суммарный ранг всех уровней ограничен сверху числом `mN` и не может убывать бесконечно. Поэтому алгоритм останавливается. В точке остановки ни один уровень не имеет допустимой редукции при текущем пороге; следовательно, дальнейшая редукция нарушает ограничение `Delta_max`. ∎

## Что говорить научному руководителю / оппоненту

В работу добавляются шесть проверяемых результатов:

1. существование и единственность композиции при допустимом согласовании;
2. псевдометрика `d_E` на пространстве объяснений;
3. верхняя оценка потери для длинных цепочек;
4. полнота и минимальность иерархии нечётких представлений;
5. аппроксимационная оценка редукции;
6. завершимость итеративной редукции `F_ML`.

Эти теоремы не требуют переписывать архитектуру: они формализуют уже существующие операции в коде.

## Про топосы, гомологии и HoTT

Топосную теорию, persistent homology и homotopy type theory лучше не вставлять как доказанные результаты текущей диссертации без отдельного аппарата, определений и источников. Без этого такие заявления выглядят сильнее, чем доказательная база.

Корректный вариант: вынести их в раздел будущих исследований:

- `Expl` как категория объяснительных объектов и допустимых согласований;
- устойчивые топологические инварианты как дополнительный диагностический признак;
- гомотопическая эквивалентность объяснений как возможное усиление `d_E`.

Так мы сохраняем математическую строгость и не даём оппоненту лёгкую точку атаки.
