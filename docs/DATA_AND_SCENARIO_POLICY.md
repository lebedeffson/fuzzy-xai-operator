# Data And Scenario Policy

HYBRID-XIRIS is a control scenario for verifying the FuzzyXAI operator route.

The values `612 / 201 / 187` and `168 / 0` belong to the control run. They are not a claim about the accuracy, industrial certification, or clinical/biometric validity of a production system.

FuzzyXAI verifies preservation of explanation channels, source-aware diagnostic conflict handling, reduction control, risk-observer action selection, proof package reproducibility, and table/report consistency.

Transfer to an industrial biometric contour requires separate domain validation, locked datasets, locked model versions, and an external certification protocol.
