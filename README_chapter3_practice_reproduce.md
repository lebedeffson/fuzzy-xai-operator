```bash
make chapter3-practice-all
```

Результат:

- `reports/chapter3_practice/package_validation_report.md`

Ограничение: Breast Cancer Wisconsin используется как реальная табличная XAI-задача, а не как клиническая апробация. `conflict-enriched` поток не трактуется как частота конфликтов в реальной практике; для обычного validation/test потока отдельно сформирован `natural flow`.
